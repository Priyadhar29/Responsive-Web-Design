Introduction

JavaScript is a high-level, interpreted programming language primarily used for web development.

It allows you to add interactivity to your web pages and is an essential part of modern web technologies.

JavaScript is versatile and widely supported by all major browsers.

  • Client-side scripting
  • Dynamic content updates
  • Event handling
console.log("Hello, World!");
Variables

Variables are containers for storing data values in JavaScript.

You can declare variables using var, let, or const.

The difference between these keywords relates to scope and mutability.

  • var - function scoped
  • let - block scoped
  • const - block scoped, immutable
let name = "Alice"; const age = 25;
Data Types

JavaScript supports different data types:

They can be categorized into primitive and reference types.

  • String
  • Number
  • Boolean
  • Object
  • Undefined
let isActive = true; let score = 99;
Functions

Functions are blocks of reusable code designed to perform a particular task.

They can take input parameters and return values.

You can define functions in multiple ways.

  • Function declaration
  • Function expression
  • Arrow function
function greet(name) { return "Hello " + name; }
Loops

Loops allow you to execute a block of code repeatedly based on a condition.

Common loop types in JavaScript:

  • for loop
  • while loop
  • do...while loop
for(let i = 0; i < 5; i++) { console.log(i); }
